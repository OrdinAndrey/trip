## Приложение для поиска маршрута путешествия

<h3 align="center">Triplify</h3>

<p align="center">
  Позволяет быстро найти ближайший маршрут между двумя населенными пунктами для поездки на автомобиле.
  <br></p>  

## Технологии
<ul>
<li>React</li>
<li>Redux toolkit</li>
<li>Typescript</li>
<li>SCSS</li>
</ul>
